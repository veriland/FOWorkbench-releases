# FinOps Workbench

Public installer downloads for **FinOps Workbench** — a desktop app for managing
Microsoft Power Platform and Dynamics 365 Finance & Operations environments.

➡ **Downloads:** https://github.com/veriland/FOWorkbench-releases/releases/latest

This "Source code" archive is intentionally empty of application code. FinOps
Workbench's source is maintained in a separate, private repository; this
repository only hosts the published installers.
